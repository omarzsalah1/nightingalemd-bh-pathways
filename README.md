# NightingaleMD Behavioral Health Triage Pathway Library

Interactive single-file HTML reference for behavioral health and crisis telephone triage on a nurse line. Sample shared with Isaac Health.

## What's inside

- 26 clinical pathways across 6 categories (suicide & self-harm, acute crisis, substance use, eating/behavioral, domestic & abuse, special populations)
- Live Mermaid SVG flowcharts, color-coded by acuity (911/ED/urgent/callback/home)
- Interactive scoring instruments: C-SSRS, PHQ-9, EPDS, GDS-15
- Sticky sidebar navigation with category grouping
- Persistent crisis-line footer (988, Crisis Text Line, SAMHSA, Poison Control)
- Embedded Mermaid.js — works fully offline

## Deployed

Live at: https://nightingalemd-bh-pathways.vercel.app (after Vercel import)

## File

- `index.html` — full interactive document (3.4 MB, self-contained, all dependencies bundled)

## Clinical review note

All pathways are paraphrased from cited primary sources (Columbia Lighthouse Project, SAMHSA, ASAM, RAINN, NEDA, AAP, ACOG, VA PTSD Center, AFSP, NIMH, AAACN). A licensed clinical team should review before production deployment.
